# computer-music

Let your computer compose music (random noise) for you!

## Requirements

- python 3.7
- python-pip
- python-tk
- virtualenv

## Run

```shell
virtualenv venv
source venv/bin/activate
pip install -r requirements.txt
python create-computer-music.py
```

## Help
```shell
python create-computer-music.py --help
```
